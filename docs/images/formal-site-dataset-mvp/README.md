# Formal Site Dataset MVP Artifact

This directory stores the separate dataset-enabled review artifact for the
formal site dataset integration MVP.

- primary artifact: `site-preset-dataset-enabled.png`
- capture command: `npm run capture:site-dataset`
- separation rule:
  - not part of `docs/images/phase-2.12/`
  - not part of the OSM Buildings showcase line
  - not part of the Phase 3.1 HUD follow-up line

This artifact proves the configured `site` preset hook can render a dataset-
backed building model through Cesium's native 3D Tiles runtime. It does not
claim that the final delivery AOI has already been supplied.
