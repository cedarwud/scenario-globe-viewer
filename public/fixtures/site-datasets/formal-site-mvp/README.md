# Formal Site Dataset MVP Fixture

This fixture is a repo-owned 3D Tiles validation asset for the formal site dataset
integration MVP.

- source: copied from Cesium `Apps/SampleData/Cesium3DTiles/Tilesets/Tileset`
- purpose: prove the configured `VITE_CESIUM_SITE_TILESET_URL` path through the
  native `Cesium3DTileset.fromUrl(...)` runtime
- non-goals:
  - it is not part of the Phase 2.12 dormant baseline
  - it is not the OSM Buildings showcase
  - it is not a claim that the final production site AOI has already been supplied
